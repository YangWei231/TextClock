# ![ic_launcher.webp](app/src/main/res/mipmap-hdpi/ic_launcher.webp) TextClock
数字时钟显示，含专注模式和番茄计时功能。

功能：
- 浅色模式和深色模式时间显示
- 番茄时钟计时。25分钟番茄计时，专注模式(深色模式)；5分钟休息计时，休闲模式(浅色模式)。
